REPORT Approval indicator
============================

A Moodle admin report. It show a list of courses and the enrolled users percent that completed the course.

Package tested in: moodle 3.3+

QUICK INSTALL
==============
Download zip package, extract the approvalindicator folder and upload this folder into report/.
